Introduction¶

Python and R have been two popular programming languages for data scientists. However, data visualization in Python has long been refered to as painful and ugly compared to the handy tools available in R. This tutorial talks about how to use ggplot for visualization in Python. We will introduce common functions with examples that have been covered in previous lectures of 5702 but in Python this time.
